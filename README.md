# Portfolio App

A full-stack stock portfolio monitoring application built with SvelteKit, FastAPI, and Supabase.

## Features

- Track stock holdings and portfolio performance
- Connect brokerage accounts via Plaid integration
- View financial news summaries powered by Claude AI
- Secure authentication with Supabase
- Responsive modern UI with TailwindCSS

## Tech Stack

- **Frontend**: SvelteKit + TailwindCSS
- **Backend**: FastAPI (Python)
- **Database & Auth**: Supabase
- **Account Integration**: Plaid API
- **AI Summaries**: Claude API
